## 车辆接入车联网并上传数据

本实验描述如何将车辆接入车联网后台. AWS IoT使用证书进行双向认证, 车辆接入前需要配置证书和策略,并将证书下载至车端.

### 1. 创建事物, 证书和策略

- iot core上生成证书, 策略

### 2. 下载证书

- 从iot core上下载相关证书
- 下载rootca

### 3. 安装SDK

- ec2上安装python3/sdk

### 4. 接入车联网并开始上传数据

- 运行car_publish.py
- 在iot console上查看上传的数据
